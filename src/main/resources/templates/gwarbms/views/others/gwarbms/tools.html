Tab操作

刷新当前Tab页面：点击刷新

// html
点击刷新
// js
$('a').on('click', function () {
    gwarbms.view.tab.refresh();
}

切换到指定Tab页面：点击切换到首页

// html
点击切换
// js
$('a').on('click', function () {
    gwarbms.view.tab.change("/index");
}

关闭指定Tab页面：点击关闭当前Tab页，并跳转到首页

// html
点击切换
// js
$('a').on('click', function () {
    var currentUrl = window.location.hash.replace('#', '');
    gwarbms.view.tab.del(currentUrl);
    gwarbms.view.tab.change("/index");
}
更多Tab的操作可以阅读static/gwarbms/lay/modules/view.js源码。

gwarbms.routeLeave(function (route, nextPath, next) {
